## Text Input Variants

### Basic Text Input
Simple text field with label and hint:
```dart
HuxInput(
  label: 'Email',
  hint: 'Enter your email address',
)
```

### Input with Icon
Text field with prefix icon:
```dart
HuxInput(
  label: 'Search',
  hint: 'Search for products...',
  prefixIcon: Icon(FeatherIcons.search),
)
```

### Input with Validation
Text field with form validation:
```dart
HuxInput(
  label: 'Password',
  hint: 'Enter your password',
  prefixIcon: Icon(FeatherIcons.lock),
  obscureText: true,
  validator: (value) {
    if (value == null || value.isEmpty) {
      return 'Password is required';
    }
    if (value.length < 6) {
      return 'Password must be at least 6 characters';
    }
    return null;
  },
)
```

## Date Input Variants

### Basic Date Input
Specialized date picker with calendar integration:
```dart
HuxDateInput(
  label: 'Birth Date',
  hint: 'MM/DD/YYYY',
  onDateChanged: (date) {
    print('Selected: $date');
  },
)
```

## Form Control Variants

### Checkbox
Interactive checkbox with label:
```dart
HuxCheckbox(
  value: isAgreed,
  onChanged: (value) {
    setState(() {
      isAgreed = value ?? false;
    });
  },
  label: 'I agree to the terms',
)
```

### Switch
Toggle switch component:
```dart
HuxSwitch(
  value: isEnabled,
  onChanged: (value) {
    setState(() {
      isEnabled = value;
    });
  },
)
```

## HuxInput

```dart
HuxInput(
  label: 'Email',
  hint: 'Enter your email address',
  prefixIcon: Icon(FeatherIcons.mail),
  validator: (value) {
    if (value == null || value.isEmpty) {
      return 'Email is required';
    }
    return null;
  },
)
```

### Properties

- `label` - Field label text
- `hint` - Placeholder text
- `prefixIcon` - Icon displayed at the start
- `validator` - Form validation function
- `controller` - Text editing controller
- `onChanged` - Value change callback

## HuxDateInput

Specialized date input with automatic formatting and integrated calendar picker.

```dart
HuxDateInput(
  label: 'Select Date',
  hint: 'MM/DD/YYYY',
  onDateChanged: (date) {
    print('Selected date: $date');
  },
  validator: (date) {
    if (date == null) return 'Date is required';
    return null;
  },
)
```

### Properties

- `label` - Field label text
- `hint` - Placeholder text (defaults to MM/DD/YYYY)
- `onDateChanged` - Date selection callback
- `validator` - Form validation function
- `showCalendarIcon` - Whether to show calendar icon (default: true)
- `format` - Custom date format (defaults to MM/dd/yyyy)
- `firstDate` - Earliest selectable date
- `lastDate` - Latest selectable date

## HuxCheckbox

Interactive checkbox with smooth animations and consistent styling.

```dart
HuxCheckbox(
  value: isChecked,
  onChanged: (value) {
    setState(() {
      isChecked = value ?? false;
    });
  },
  label: 'I agree to the terms and conditions',
)
```

### Properties

- `value` - Current checked state
- `onChanged` - State change callback
- `label` - Optional label text

## HuxSwitch

Toggle switch for binary choices with smooth animations.

```dart
HuxSwitch(
  value: isEnabled,
  onChanged: (value) {
    setState(() {
      isEnabled = value;
    });
  },
)
```

### Properties

- `value` - Current switch state
- `onChanged` - State change callback
